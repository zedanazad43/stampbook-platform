API Endpoints

Wallet

POST /api/wallet/create Create a new wallet
GET /api/wallet/:userId Get wallet details
POST /api/wallet/transfer Transfer balance between wallets
GET /api/wallet/:userId/transactions Get transaction history

Market

GET /api/market/items List all market items
POST /api/market/items List a new item for sale
POST /api/market/items/:itemId/buy Purchase a market item

Blockchain

GET /api/blockchain/info Get token metadata
GET /api/blockchain/supply Get token supply stats
POST /api/blockchain/mint Mint STP tokens (auth required)
