🏷️ Stampcoin Platform

Blockchain-inspired digital stamps — wallet, marketplace & STP token

💼 Digital Wallet

Create and manage digital wallets, transfer STP balances, and store your stamp collection securely.

🛒 Marketplace

List, browse, and purchase digital stamps. Full transaction history with buyer and seller filtering.

⛓️ Blockchain

BEP-20-compatible STP token on BNB Smart Chain. Track supply, mint events, and address balances.

Wallet API

MethodPathDescription
POST/api/wallet/createCreate a new wallet
GET/api/wallet/:userIdGet wallet by user ID
POST/api/wallet/transferTransfer balance between wallets
GET/api/wallet/:userId/transactionsGet transaction history
POST/api/wallet/:userId/stampsAdd stamp 🔒 auth
GET/api/walletsList all wallets 🔒 auth
POST/api/wallet/:userId/topupTop up balance 🔒 auth

Market API

MethodPathDescription
GET/api/market/itemsGet all items (supports ?status=, ?type=)
POST/api/market/itemsList a new item
GET/api/market/items/:itemIdGet item by ID
PUT/api/market/items/:itemIdUpdate item (seller only)
POST/api/market/items/:itemId/buyPurchase an item
DELETE/api/market/items/:itemIdRemove item (seller only)
GET/api/market/transactionsGet transaction history

Blockchain API

MethodPathDescription
GET/api/blockchain/infoGet token metadata
GET/api/blockchain/supplyGet token supply stats
POST/api/blockchain/mintMint STP tokens 🔒 auth
GET/api/blockchain/balance/:addressGet address balance
GET/api/blockchain/mint/eventsGet mint audit log 🔒 auth
