🪙 Stampcoin Platform

A blockchain-inspired digital stamps platform with wallet, marketplace, and BEP-20 token features.

v1.0.0 · Live on GitHub Pages

💼 Digital Wallet

Create wallets, transfer balances, manage stamps, and view full transaction history.

🛒 Marketplace

List, buy, update, and remove digital stamp items with full transaction tracking.

⛓️ Blockchain

BEP-20-compatible STP token with supply tracking, minting, and per-address balances.

Key API Endpoints

  • POST/api/wallet/create
  • GET/api/wallet/:userId
  • POST/api/wallet/transfer
  • POST/api/market/items
  • GET/api/market/items
  • POST/api/market/items/:itemId/buy
  • GET/api/blockchain/supply
  • POST/api/blockchain/mint
